# Front End Books

记录和推荐一些认为还不错的前端方面的书籍。列入标准：

 1. 看过，并觉得值得推荐给大家的。
 2. 正在看的，已经觉得不错的，或者听闻别人说不错的。
 3. 打算看的，经多人推荐，一致认为必看的。

> 注：此列表仅做推荐只用，并非评判书籍的优劣。大致依据个人喜好程度排序，并尽量客观。

如果有你觉得不错的书籍我这没有列出，欢迎将其推荐给我，也推荐给大家。
